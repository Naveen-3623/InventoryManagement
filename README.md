Project: InventoryManagementSystem

Overview:

The Inventory Management System is a Java console application that manages product stock, handles purchase and sales operations, and prevents negative inventory using custom exceptions.

Technologies Used:

Java,

OOP Concepts,

Collections (ArrayList),

Custom Exception Handling.

Features:

✔ Add Product Stock (Purchase)

✔ Sell Product Stock

✔ Prevent Negative Inventory

✔ Validate Product ID

✔ Maintain Transaction History

✔ Print Stock Summary

Project Structure:

entity – Product & Transaction classes

service – Business logic (InventoryService)

util – Custom Exceptions

main – Application Entry Point

How to Run:

Open project in Eclipse or any Java IDE.

Compile all classes.

Run Main.java.
